# Waveform-Plotting-and-Unit-conversion
### The remove_response function in Obspy can combine station data with raw waveform data to convert values between different units (e.g., converting counts to displacement or velocity). <br>
* The parameters could be: <br>
* filterBand=(0.5, 0.8, 20.0, 25.0) <br>
* level=60 <br>
